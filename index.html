How To Play

Aim: drag from the white cue ball. Longer pull means more power.

Power: green is light, yellow is medium, red is max. Controlled shots get a tiny pocketing assist.

Score: pocket all colored balls. Scratches add +1 shot.

Bonus: glowing pockets match a ball color. Sink that color there for -1 shot.

Orbs: collect Accuracy for blue rail-bounce aim or Bumper to place a one-shot pinball wall.

Daily: the layout changes each day using the date as the seed.

Practice: start a random seed or paste a friend's seed to play the same table.
